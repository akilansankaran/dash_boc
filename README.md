# Immigration Tracker — Charles River Economics Labs

This project is an interactive immigration analysis tracker developed by **Charles River Economics Labs** (CREL) in conjunction with the **Bank of Canada**, with institutional support from **Harvard University**. It provides a comprehensive comparative analysis of immigration inflows across the United States and Canada.

## Pages

- **Home (`index.html`)** — Landing page with an interactive flowchart of the analysis framework and navigation cards linking to each section.
- **Cross-Country Comparison (`comparison.html`)** — Side-by-side analysis of admission volumes, labor market integration, fiscal impact, demography, and policy architecture across both countries.
- **United States (`us.html`)** — U.S.-specific analysis covering admission flows, visa categories, economic impact, geography, and policy timeline.
- **Canada (`canada.html`)** — Canada-specific analysis covering admission flows, Express Entry, economic integration, provincial data, and demographics.

## Search

A site-wide search feature indexes all figures, charts, and analysis sections across every page. Our search feature is accessible from the search bar on any page and supports navigation directly to specific figures via deep linking.
